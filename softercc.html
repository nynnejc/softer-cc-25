setup()

p5.js has its own syntax and built-in methods that we will usually always apply to our drawing. We know that something is a method whern we see it ending with()

One of these is called setup()

Here we put any code that is involved with setting up the initial state of your program: size of you graphics, canvas, the weight of your stroke or the speed of your program.

Example:

function setup() {

    createCanvas(400, 200); // Setting up size of our drawing

}

draw()

draw() is where all the code goes that is involved with drawing to the screen: setting background color, drawing shapes etc.

Example:

function draw() {

    background(220); // setting a background color (gray)

}

Notice how each method takes arguments inside the (). For example here:

ellipse(50, 100, 50);

Go to the Ellipse documentation to find out what each number means, or try it out yourself to see if you can make the circle bigger, smaller, place it in a different spot ot change its color.
